Une offre logicielle maintenue et exploitée par un fournisseur tiers.

<!--more--> 

Les services gérés incluent, par exemple, AWS EC2, Azure SQL Database ou GCP Pub/Sub.  
De manière générale, il s’agit de tout service logiciel pouvant être utilisé par une application sans que l’utilisateur ait à en gérer l’infrastructure ou la maintenance.
